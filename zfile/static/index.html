z.rslv.xyz is a resolver for files in published Zenodo packages.

Usage:

https://z.rslv.xyz/DOI[/file]

DOI = published DOI without scheme, url, or other embellishments. e.g. "10.5281/zenodo.11400483"
file = Name of file in published package.

If file is not provided, then the the package JSON is returned from the Zenodo API.
Source
https://github.com/datadavev/zfile
